Report Monitoring & Auto-Alerts
How to stay current as new CBRE quarterly reports publish

The Pattern

CBRE publishes all Industrial Figures reports at predictable URLs: cbre.com/insights/figures/[market-name]-industrial-figures-q[N]-[YEAR]. New reports typically appear 3–4 weeks after each quarter ends. Q2 2026 reports are expected in early July 2026.

Option 1 — Bookmark the CBRE Hub

Bookmark cbre.com/insights/us-quarterly-figures — CBRE's central hub. Check it in the first 2 weeks of April, July, October, and January each year.

Option 2 — Automated GitHub Actions Script

  • A Python script runs on a schedule inside your GitHub repository — no server needed.
  • Checks all 11 market URLs monthly, detects when new quarterly reports go live.
  • Sends an email alert to your distribution list and can automatically update this website.
  • 100% free using GitHub Actions.
Microsoft 365 Users: Power Automate (included in most M365 plans) can monitor a URL and send a Teams/email alert with zero custom code. Setup time: ~30 minutes.
What's Missing — Recommendations
Additional data that would strengthen this document for investor and executive use

1. Cap Rate & Investment Sales Data

CBRE publishes a separate Industrial Cap Rate Survey quarterly. Including market-level cap rates (~5.0%–5.8% for Class A nationally) would give investors a direct valuation benchmark.

2. 3-Year Rent Growth Comparison Table

A rent growth comparison table across all 11 markets (3-year CAGR) would be a compelling investor visual showing which markets have delivered the strongest compounding rent growth — a key proxy for asset appreciation.

3. Tariff / Trade Risk Matrix

A simple matrix rating each market's sensitivity (High/Medium/Low) to import tariff risk — based on port dependence vs. domestic distribution orientation. CBRE already flags Louisville and Nashville as "insulated"; building this out for all 11 markets would be highly differentiated.

4. Power & Infrastructure Availability

Data center demand is creating power constraints in Columbus, Phoenix, and Chicago. Available power capacity (MW) is now a site selection criterion as important as clear heights.

5. Labor Market Snapshot

Industrial occupiers rank labor availability as their #1 site selection factor. A per-market table of logistics workforce size, warehouse worker wages vs. national average, and union rate would complete the investor picture.
